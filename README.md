# dotfiles

I put dotfiles here that I don't need to separate.

* ./etc/ -> /etc/
* ./home/user/ -> /home/$USERNAME/

* ./git-hooks/: my git hooks
* ./mozc/: my mozc/Google Japanese Input files
* ./windows/: windows config files
