# Markdown

## 強調は控える

以下の記号などで使える強調構文は控えてください。

- `*`
- `**`
- `_`
- `__`

今のLLMのデフォルト出力では、
強調する意味のない部分にも強調シンタックスが乱用されています。

またClaudeやChatGPTやGeminiやGrokなどのフロントエンドのレンダラーは、
強調シンタックスを正確に処理しないことが多いため、
表示がしばしば壊れます。
特に日本語が交じると表示が壊れることが多いです。

強調しなくても意味が十分に通じる場合は、
強調シンタックスは使わないでください。

例外として、

- 強調することが意味的に重要
- 文章テンプレートで強調構文を使うことを要求されている

などの場合は許可されます。
